# Inter font files go here (Phase 5e)

Add the following .ttf files to this directory; the bundle id is
`xyz.speakeasyapp.app` so React Native's autolinking picks them up by filename:

  Inter-Light.ttf
  Inter-Regular.ttf
  Inter-Medium.ttf
  Inter-SemiBold.ttf
  Inter-Bold.ttf

(iOS equivalent: add the same .ttf files to ios/Speakeasy/, reference in
ios/Speakeasy/Info.plist under UIAppFonts. iOS work is queued — see spec §11
Phase 5a.)
